# Aurekai MCP Server

Minimal Aurekai MCP server scaffold for registry submission.

## Install

```bash
npm install -g @aurekai/mcp
```

## Commands

- `aurekai-mcp help`
- `aurekai-mcp doctor`
- `aurekai-mcp manifest`
